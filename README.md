## Overview

This repository contains all assignments for the **Data Analysis** course. Each module is organized in its own folder, containing:

* **Dataset(s)** – raw or cleaned data for analysis
* **R scripts** – code files for data preprocessing, analysis, and visualization
* **Output files** – tables, CSVs, and graphs generated during analysis
* **README.md** – description of the module, methods used, and key results

This structure ensures clarity, reproducibility, and easy navigation throughout the semester.

---

> Each module folder is self-contained with all necessary files for that assignment.

---

## Technologies & Tools

* **Programming Language:** R (version 4.5)
* **IDE:** RStudio (Windows)
* **File Formats:** `.csv`, `.R`, `.png`, `.pdf` for visual outputs
